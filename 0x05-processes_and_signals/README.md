# 0x05. Processes and signals
## About
An introductory project on:
- What is a PID
- What is a process
- How to find a process' PID
- How to kill a process
- What is a signal
- What are the 2 dignals that cannot be ignored

## Requirements
- Python 3.4
- Shellcheck 0.3
